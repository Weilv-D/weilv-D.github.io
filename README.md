# 未来回忆录

Weilv 的个人博客，记录技术与生活。

基于 [Astro](https://astro.build) 构建，追求极简与高性能。

## 技术栈

- **框架**: Astro v4
- **样式**: Tailwind CSS
- **交互**: React
- **部署**: GitHub Pages

## 本地运行

```bash
git clone https://github.com/Weilv-D/WeilvBlog.git
cd WeilvBlog
npm install
npm run dev
```

访问 `http://localhost:4321/WeilvBlog/`

## 写作

文章位于 `src/content/blog/`。

```markdown
---
title: '标题'
description: '描述'
pubDate: '2025-12-25'
category: 'Tech'
tags: ['tag1', 'tag2']
---
```
