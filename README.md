# 未来回忆录

这是 [Weilv](https://github.com/Weilv-D) 的个人博客源码仓库。

访问地址：[https://weilv.space/](https://weilv.space/)

## 关于

记录技术探索的足迹与生活中的吉光片羽。

### 设计理念

- **未来回忆录风格**：全站采用 **Bodoni Moda** 高对比度衬线字体，营造优雅、锐利的视觉节奏。
- **液态玻璃质感**：Squircle 圆角与磨砂玻璃特效的现代融合。

基于 [Astro](https://astro.build) 构建，追求极致性能与阅读体验。

## 开发

```bash
npm install
npm run dev
```

## 部署

推送到 `main` 分支自动触发 GitHub Actions 部署。
