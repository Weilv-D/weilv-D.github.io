## 为什么不是只换视觉

如果只是把背景换成旧纸纹理、标题换成衬线字、再加几条装饰线，这更像是“哈利波特主题皮肤”，而不是《预言家日报》式的排版升级。

真正的差异在于版面逻辑：

- 头版标题需要能根据宽度自动拟合。
- 文字要能围绕人物、插画、广告块和引文框流动。
- 多栏不是把内容切成两段，而是一段连续文本跨栏交接。
- 当窗口尺寸变化时，整个版心要迅速重排，而不是抖动、跳高、塌陷。

## 为什么用 Pretext

`pretext` 给的不是现成 UI，而是一套更关键的能力：**不用读 DOM 布局，就能知道文本如何断行、会占多高、每一行在哪里结束**。

<ProphetPullquote quote="报纸感最迷人的地方，不是复古，而是文字和图像之间那种持续谈判的紧张关系。" cite="专题编辑注" />

这让博客里本来过于昂贵的排版效果突然变得实际：

- 文章列表可以做真正稳定的变体卡片高度预测。
- 专题页可以让引文框和人物插图实时参与正文排版。
- 章节页可以做多栏杂志式布局，而不是依赖脆弱的 CSS hacks。

## 这次原型的定位

这不是最终版，而是第一块基座：

1. 新增一套 `prophet` 布局开关。
2. 建立报纸主题 token 和独立文章模板。
3. 在文章顶部放入一个 `pretext` 驱动的 editorial island。
4. 保留下方标准 prose 正文，避免长文阅读体验倒退。
